# HomePage 리디자인 스펙

## 목표

Figma 모바일 UI 템플릿(Mobile app UI Template) 스타일을 Petory HomePage에 적용.
디자인만 교체하며 App.js 탭 시스템 및 기존 라우팅 구조는 변경하지 않는다.

## 범위

- **변경**: `frontend/src/components/Home/HomePage.js`
- **불변**: `App.js` 탭 시스템, 기존 각 도메인 페이지들, `theme.js`

## 탭별 데이터 매핑

| 탭         | 히어로 카드                   | 가로 스크롤 카드 | API                  |
| ---------- | ----------------------------- | ---------------- | -------------------- |
| 주변서비스 | 평점 높은 서비스 1개          | 인기 서비스 목록 | `locationServiceApi` |
| 모임       | 모집 중 모임 1개              | 최신 모임 목록   | `meetupApi`          |
| 실종신고   | 최근 조회수 높은 실종글 1개   | 최근 실종 목록   | `missingPetApi`      |
| 커뮤니티   | 인기 게시글 1개 (자랑글 우선) | 인기 게시글 목록 | `boardApi`           |

각 탭 최초 진입 시 API 호출. 이후 탭 재진입은 캐시(useState) 재사용.

## 디자인 토큰

Figma → 기존 `theme.js` 매핑:

| 역할                  | 값                                                               |
| --------------------- | ---------------------------------------------------------------- |
| 히어로 카드 배경      | 이미지 + `rgba(0,0,0,0.4)` 오버레이                              |
| 글래스 패널           | `backdrop-filter: blur(20px)`, `background: rgba(29,29,29,0.35)` |
| 카드 border-radius    | `theme.borderRadius['2xl']` (24px)                               |
| 히어로 카드 높이      | `280px`                                                          |
| 가로 스크롤 카드 크기 | `160px × 200px`                                                  |
| 탭 활성 색상          | `theme.colors.primary` (#E8714A)                                 |
| 검색바                | `theme.colors.border` 테두리, `theme.borderRadius.pill`          |

히어로 이미지 없을 시: 도메인 색상(`theme.colors.domain.*`) 그라디언트로 대체.

## 컴포넌트 구조

```
HomePage
  ├── HomeHeader          (아바타, 인사, 알림 아이콘)
  ├── SearchBar           (더미 UI, 클릭 시 추후 기능 연결)
  ├── CategoryTabs        (주변서비스 / 모임 / 실종 / 커뮤니티)
  └── TabContent          (activeTab에 따라 렌더)
        ├── HeroCard      (대형 이미지 카드 + 글래스 패널)
        └── HorizontalCardList  (가로 스크롤 카드 리스트)
```

모두 `HomePage.js` 단일 파일 내 styled-components로 구현. 별도 파일 분리 없음.

## 에러/로딩 처리

- 로딩: 히어로 카드 자리에 스켈레톤 (회색 블록)
- API 실패: 히어로 카드 숨김, 가로 카드 빈 상태 메시지 ("아직 등록된 항목이 없어요")
- 이미지 로드 실패: 도메인 그라디언트 폴백

## 제약 조건

- `setActiveTab` prop 유지 (App.js 인터페이스 불변)
- `useAuth` hook 유지 (닉네임, 관리자 여부)
- 관리자 섹션은 제거하지 않고 하단에 유지 (리디자인된 카드 스타일로)
- styled-components 사용 (기존 패턴 유지)
