# 위치 기반 서비스 알고리즘

## 1. Haversine 거리 계산

**목적**: 두 좌표(위도/경도) 간 거리를 미터 단위로 계산

**위치**: `LocationServiceService.calculateDistance()`

**공식**:
```
R = 6371000 (지구 반경, 미터)
dLat = rad(lat2 - lat1)
dLng = rad(lng2 - lng1)
a = sin²(dLat/2) + cos(lat1)·cos(lat2)·sin²(dLng/2)
c = 2·atan2(√a, √(1-a))
distance = R × c
```

**코드**:
```java
final int R = 6371000; // 지구 반경 (미터)
double dLat = Math.toRadians(lat2 - lat1);
double dLng = Math.toRadians(lng2 - lng1);
double a = Math.sin(dLat/2)*Math.sin(dLat/2) +
    Math.cos(Math.toRadians(lat1))*Math.cos(Math.toRadians(lat2))*
    Math.sin(dLng/2)*Math.sin(dLng/2);
double c = 2*Math.atan2(Math.sqrt(a), Math.sqrt(1-a));
return R * c; // 미터
```

## 2. 지역 계층 검색 우선순위

**목적**: 사용자가 입력한 지역 파라미터에 따라 검색 범위 결정

**위치**: `LocationServiceService.searchLocationServicesByRegion()`

**우선순위** (좁은 범위 → 넓은 범위):
1. `roadName` (도로명) → `findByRoadName`
2. `eupmyeondong` (읍면동) → `findByEupmyeondong`
3. `sigungu` (시군구) → `findBySigungu`
4. `sido` (시도) → `findBySido`
5. 전체 → `findByOrderByRatingDesc`

**추가 필터**: `category`가 있으면 인메모리 필터링 (category1/2/3 매칭)

## 3. 반경 검색 (ST_Distance_Sphere)

**목적**: 사용자 위치 기준 반경 내 서비스 검색

**위치**: `SpringDataJpaLocationServiceRepository.findByRadius()`, `findByRadiusOrderByDistance()`

**2단계 전략**:
1. **1차**: MBR(Minimum Bounding Rectangle)로 공간 인덱스 활용
   - `ST_Within(location, POLYGON(...))` — 위도/경도 범위로 반경을 사각형으로 근사
   - `1도 ≈ 111km`, 경도는 `111 * cos(radians(lat))`로 보정
2. **2차**: 정확한 거리 계산
   - `ST_Distance_Sphere(location, POINT(lat, lng), 4326) <= radiusInMeters`

**정렬**: `findByRadiusOrderByDistance`는 거리 오름차순 정렬

## 4. 키워드 검색 + 거리 계산

**위치**: `LocationServiceService.searchLocationServicesByKeyword()`

**흐름**:
1. DB에서 키워드 검색 (주소/시도/시군구/읍면동 포함)
2. 사용자 좌표가 있으면 → `calculateDistance()`로 각 결과에 거리 추가
3. `maxResults`로 결과 수 제한
