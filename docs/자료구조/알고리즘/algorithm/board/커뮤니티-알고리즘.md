# 커뮤니티 & 실종 제보 알고리즘

## 1. 인기 게시글 점수 (BoardPopularityService)

**목적**: "자랑" 카테고리 게시글의 인기도 순위 산정

**점수 공식**:
```
score = (likes × 3) + (comments × 2) + views
```

**위치**: `BoardPopularityService.calculatePopularityScore()`

**동점 시**: `createdAt` 내림차순 (최신 우선)

---

## 2. 스냅샷 조회 우선순위

**목적**: 기간별 인기 게시글 스냅샷 조회 (WEEKLY / MONTHLY)

**위치**: `BoardPopularityService.getPopularBoards()`

**우선순위**:
1. **정확한 날짜 매칭**: `periodType` + `periodStartDate` + `periodEndDate` 일치
2. **기간 겹침**: `periodStartDate <= 조회종료일 AND periodEndDate >= 조회시작일`
3. **최근 스냅샷**: `findTop30ByPeriodTypeOrderByPeriodEndDateDescRankingAsc`
4. **없으면 생성**: `generateSnapshots()` — 실시간 집계 후 저장

---

## 3. 기간 범위 계산 (PeriodRange)

| periodType | periodStart | periodEnd |
|------------|-------------|-----------|
| WEEKLY | today - 6일 | today |
| MONTHLY | today - 29일 | today |

---

## 4. 배치 집계 (generateSnapshots)

**흐름**:
1. "자랑" 카테고리 게시글 조회 (레거시 호환: "PRIDE"로도 시도)
2. `getLikeCountsBatch`, `getCommentCountsBatch`, `getViewCountsBatch` — IN 절 1000개 단위 배치
3. 각 게시글별 점수 계산 → 정렬 → 상위 30개
4. 기존 스냅샷 삭제 후 새로 저장

---
