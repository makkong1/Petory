# 🐾 Petory 플젝

- **플젝명** : 종합 반려동물 케어 & 정보 & 커뮤니티 플랫폼  
- **목적** : 늘어난 애완동물 보호자들을 위한 웹서비스  

---

## 🎯 목표 기능

### 1️⃣ 지역 서비스 정보 제공 (메인 서비스)
- **기능**
  - 지도 기반 병원 / 용품점 / 유치원 위치 표시  
  - 필터 : 거리, 평점, 가격대  
  - 리뷰 / 댓글 기반 추천  

---

### 2️⃣ 펫시터 / 케어 매칭 서비스
- **기능**
  - 유저가 “하루/이틀 케어가 필요하다” 글을 등록  
  - 케어 제공자가 신청글을 남김 → 댓글/신청 수락 후 대화 시작 (소켓으로 실시간 소통) → 거래 진행  
- **추가 아이디어**
  - 거래 종료 후 평점 / 리뷰 기능  
  - 제공자의 예약 달력 / 일정 표시  
  - 펫시터 신고 기능  

---

### 3️⃣ 커뮤니티
- **기능**
  - Q&A 게시판  
  - 자유게시판 등  

---

### 4️⃣ 종합 알림 / 매칭 시스템
- **기능**
  - 케어 요청자 / 제공자 매칭 알림  
  - 예약 알림 등  

---

## 🔧 추가 고려 사항
- **확장성**
  - Redis 캐시 → 인기 지역 서비스, 추천 게시물  
  - RabbitMQ → 알림 / 푸시 서비스  

---

## ⚙️ 기술 스택
- **Backend** : Spring Boot 3.X  
- **Database** : MySQL 8.0  
- **Cache** : Redis  
- **Message Broker** : RabbitMQ  
- **Frontend** : React  
- **기타** : Docker, IntelliJ IDEA, GitHub 등
