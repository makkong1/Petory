# PRD: Petory (페토리)

## 목표
데이터 기반의 반려동물 케어(매칭) 및 오프라인 커뮤니티(소모임/실종제보) 통합 플랫폼 구축.

## 핵심 기능
1. **위치 기반 서비스 (LBS)**: 사용자 반경 내 펫 동반 시설(병원, 카페) 탐색 및 하버사인 거리 계산.
2. **펫 케어 & 에스크로 매칭**: 반려동물 돌봄 요청/지원 및 플랫폼 재화(펫코인)를 통한 안전 결제.
3. **로컬 커뮤니티 & 실시간 채팅**: 실종 동물 긴급 제보, 오프라인 소모임 운영, 매칭된 사용자 간 웹소켓 채팅.

## MVP 제외 사항
- 펫 용품 이커머스(쇼핑몰) 기능 및 장바구니/배송 연동 (현재 스코프 분산 방지).
- 영상 통화 및 라이브 스트리밍 기능 (채팅 및 사진/텍스트 리뷰로 대체).
- 외부 지도 API 기반 자체 구석구석 내비게이션(길찾기) 직접 디스플레이 (단순 링크/거리 표출까지만).
